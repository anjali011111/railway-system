# 🚂 Railway Ticket Booking System

A Full Stack DBMS Project

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: MySQL

## Features
- Add and manage trains
- Register passengers
- Book and cancel tickets
- Animated cinematic railway background
- SQL Triggers, Procedures, Views

## How to Run
1. Run database/schema.sql in MySQL Workbench
2. Open backend/db.js and add your MySQL password
3. cd backend → npm install → node server.js
4. Open http://localhost:3000
